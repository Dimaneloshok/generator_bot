# Generator Bot ⚡

Telegram/WEB-приложение для подбора генератора по параметрам пользователя.

Проект помогает подобрать подходящий генератор, учитывая:
- напряжение (220В / 380В)
- подключаемые потребители
- пусковые токи
- тип запуска
- наличие АВР
- дополнительные товары (масло, колёса, ручки, удлинители)

## Возможности

✅ Подбор генератора по мощности  
✅ Учёт пусковых токов оборудования  
✅ Выбор стартера  
✅ Выбор АВР  
✅ Добавление сопутствующих товаров  
✅ Формирование заказа  
✅ Работа с локальной базой Excel

# Запуск проекта


## 1. Установка Python

Необходим Python версии 3.11+

Проверка:
python --version


## 2. Клонирование проекта
git clone https://github.com/Dimaneloshok/generator_bot.git

Перейти в папку:
cd generator_bot


## 3. Создание виртуального окружения

Windows:
python -m venv venv


Активация:

PowerShell:
venv\Scripts\activate


После активации в консоли появится:
(venv)


## 4. Установка зависимостей
pip install -r requirements.txt


## 5. Запуск приложения
python app.py


После запуска появится:
Running on http://127.0.0.1:5000
